Install jhipster-generator:
jhipster.tech website is blocked by proxy.
Workaround:
Install npm
Configure npm registry to Artifactory

npm config set registry https://artifactory.us.bank-dns.com/artifactory/api/npm/npm-remotes/

Login to Artifactory, provide your credentials.

npm login

Install jhipster-generator npm package

npm install -g generator-jhipster

Download jhipster-registry
Navigate to https://github.com/jhipster/jhipster-registry/releases
Download latest release jar
Copy it to C:\_home\programs directory
Ensure the following ENV properties are set:
Path to include C:\Users\<<NTLOGIN>>\AppData\Roaming\npm
JAVA_HOME set to a JDK > 11 and < 19
Commands:
Open PowerShell and run the below commands:

$Env:Path+=";C:\Users\FHVASA\AppData\Roaming\npm"

$Env:JAVA_HOME="C:\_home\programs\jdk-17.0.6.10-hotspot"

$Env:NODE_EXTRA_CA_CERTS = "C:\_home\programs\heroku\certs\USBankRootCA.cer"

$Env:HTTP_PROXY = "http://proxy.na.global.prv:8080"

$Env:HTTPS_PROXY = "http://proxy.na.global.prv:8080"
